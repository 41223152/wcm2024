實作練習問題

1.一份採 UTF-8 中文編碼的 .csv (例如: 5j_wcm2024_midterm.csv)在 Windows Excel 2016 開啟之後呈現亂碼, 該如何設定改為用 UTF-8 編碼?

  1. 開啟 Excel 2016：點擊開啟 Excel 2016 應用程式。
  2. 匯入 CSV 檔案：
    • 在 Excel 中選擇「資料」選項卡。
    • 點擊「從文字」或「從文字檔案」選項。
  3. 選擇 CSV 檔案：
    • 瀏覽您的電腦，找到並選擇要匯入的 CSV 檔案。
  4. 指定正確的編碼：
    • 在打開檔案的對話框中，選擇正確的檔案編碼。
    • 在「檔案來源」設定中，選擇「Unicode (UTF-8)」。
    • 確認「文件原始格式」設定為「分隔字元」，並設置正確的分隔符號（通常是逗號或分號）。
  5. 匯入 CSV 檔案：
    • 點擊「讀取」或「匯入」按鈕，將 CSV 檔案匯入 Excel。
  6. 檢查文字顯示：
    • 檢查匯入後的資料是否正確顯示中文文字，確保沒有亂碼。
  7. 儲存檔案：
    • 如果所有文字都正確顯示，您可以將檔案另存為 CSV 格式。
    • 選擇「檔案」>「另存為」，選擇 CSV 格式，並在儲存時確保選擇 UTF-8 編碼

4.為什麼 wcm2024 倉儲中的 cms.bat 在 Windows 啟動之後, 若同時多人連線, 系統會無法負荷, 同一台電腦硬體有沒有可能採其他方法啟動, 可以同時讓多人連線且不會產生延遲或停頓的問題? 例如採 wsgi 方式啟動, 這該如何進行? 當設定好讓 Flask 編寫的網際內容管理系統透過效率較高的 wsgi 模式執行後, 如何讓執行程式與電腦開機同步執行? 直接執行套件與透過 Windows services 執行有何不同? NSSM 是什麼工具?

當多個用戶同時連線到一台電腦運行的服務時，效能問題常常會出現。在 Windows 系統中，使用 .bat 檔案直接啟動服務可能會導致效能瓶頸。一種更好的方法是使用 WSGI（Web Server Gateway Interface）來啟動 Flask 應用程序。WSGI 是 Python Web 應用程序的標準接口，它允許 Web 服務器與 Python 應用程序之間進行通信，提供了更好的性能和可擴展性。要將 Flask 應用程序轉換為 WSGI 模式，你可以使用像 Gunicorn、uWSGI 或 mod_wsgi 等 WSGI 伺服器來運行你的應用程序。這些伺服器能夠處理多個同時連線，並且通常比直接運行 Flask 應用程序的方法效率更高。讓執行程式與電腦開機同步執行，你可以將其設置為作為一個服務（Service）在 Windows 啟動時自動運行。這可以通過 Windows 服務管理員來完成，或者使用特定的工具如 NSSM（Non-Sucking Service Manager）。NSSM 是一個常用的工具，用於在 Windows 上創建和管理服務。它允許你將任何可執行文件轉換為 Windows 服務，並提供了許多進階功能，如日誌記錄、重新啟動策略等。直接執行套件和透過 Windows services 執行之間的主要區別在於穩定性和管理性。使用 Windows services 可以更方便地管理服務，例如設置啟動參數、自動重啟、記錄事件等，同時使得這些服務在系統啟動時自動運行，而無需任何用戶交互。總之，對於運行 Flask 應用程序，特別是在多個用戶同時訪問時，建議使用 WSGI 伺服器來提高性能和穩定性，同時將其設置為 Windows 服務，以確保其在系統啟動時自動運行並具有更好的管理性。

5.如何在一台 Windows (Ubuntu) 電腦快速建立多人帳號? 建立多人帳號之後, 如何透過程式將帳號密碼寄給使用者?

Windows 上，你可以通過以下步驟來建立多個用戶帳號：

  1. 打開「控制台」。
  2. 選擇「用戶帳戶」。
  3. 選擇「新增或移除用戶帳戶」。
  4. 點擊「新增用戶帳戶」。
  5. 輸入新用戶的名稱。
  6. 選擇帳戶類型，可以是「管理員」或「標準使用者」。
  7. 如果需要，為新用戶設置密碼。
  8. 完成後，按「完成」。

8.假如希望直接在網頁中利用 Brython 建立一個掃地機器人的工作模擬系統, 該如何進行?

  1. 準備 HTML 文件：首先，創建一個 HTML 文件，其中包含掃地機器人模擬的介面和必要的元素。這個 HTML 文件將包含一個畫布（canvas）元素，用於顯示掃地機器人的運動軌跡和清理效果。

  2. 引入 Brython：在 HTML 文件中引入 Brython 的 JavaScript 文件，以便在網頁中運行 Python 代碼。

  3. 編寫 Brython 代碼：在 HTML 文件中編寫 Brython 代碼，來模擬掃地機器人的行為。這些代碼將控制掃地機器人在畫布上移動，以及清理地面的過程。

  4. 設計掃地機器人模擬介面：設計一個用於控制掃地機器人模擬的網頁界面，例如按鈕或滑塊，用戶可以通過這些界面來控制機器人的行動。
