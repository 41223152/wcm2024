W4

有關建立的網站：網站是指針對網頁瀏覽器檢視的內容進行管理的網站，利用Github所提供的頁面功能來建立網頁進行管理。

Replit靜態網站檢視：若要執行動態網站，可以在Shell中執行python3 main.py或使用Replit的運作按鈕，Shell執行python3 main2.py就可以執行以太坊內的靜態網站檢視。

Replit建立錢包權限：

套件網站匯入Replit，必須自行在 /home/runner/.ssh 目錄下建立 id_rsa 和 config (id_rsa.pub 必須登入 Github 對應帳號)，其中的 config 內容為:

託管 github.com
用戶git
主機名稱 github.com

ssh.py程式內容如下：

導入作業系統
如果不是 os.path.exists("/home/runner/.ssh"):
    os.makedirs(“/home/runner/.ssh”)
os.system("mv ./downloads/id_rsa.txt /home/runner/.ssh/id_rsa")
os.system("mv ./downloads/config.txt /home/runner/.ssh/config")
os.system("chmod og-rwx /home/runner/.ssh/id_rsa")
